# Scripts by Raymond Slater

This repository contains a collection of automation and integration scripts for IT administration, grouped by purpose and platform.

---

## PowerShell Scripts

PowerShell automation scripts for Windows administration, Microsoft 365, Azure, and system management tasks. Includes utilities for Exchange Online, Azure app provisioning, and more.

See [`Powershell/README.md`](Powershell/README.md) for complete documentation and usage examples.

### Quick Access
Several scripts are available via web shortcuts at `ps.cqts.com.au`:
- `/speedtest` - Ookla Speedtest CLI runner
- `/addwifi` - Wireless network configuration
- `/reckonfw` - Reckon Accounts firewall rules
- `/ods` - OneDrive sync health checker

## ITGlue API Tools

Python-based tools for auditing and extracting data from ITGlue via REST API and web scraping.

**Features:**
- Full password and folder extractor using Selenium
- Folder hierarchy resolution (not available via native API)
- Resume-safe execution with caching
- Automatic rate-limit handling

See [`ITGlue_api/README.md`](ITGlue_api/README.md) for setup and usage instructions.
